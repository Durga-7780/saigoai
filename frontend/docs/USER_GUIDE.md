## Attendance Management

### Manual Check-In

1. Go to Dashboard
2. Click "Check In" button
3. Your location will be captured automatically
4. Confirmation message will appear

### Manual Check-Out

1. Go to Dashboard
2. Click "Check Out" button
3. Total hours will be calculated
4. Confirmation message will appear

### Fingerprint Attendance

#### First Time Setup

1. Go to Profile
2. Click "Enroll Fingerprint"
3. Select finger (1-10)
4. Place finger on scanner
5. Wait for confirmation

#### Daily Use

1. Go to Attendance page
2. Click "Fingerprint Check-In"
3. Place enrolled finger on scanner
4. System will verify and mark attendance

### View Attendance History

1. Go to Attendance page
2. Select date range
3. View attendance records with:
   - Check-in time
   - Check-out time
   - Total hours
   - Status (Present/Late/Absent)

### Attendance Statistics

View monthly statistics:
- Total working days
- Present days
- Late arrivals
- Average working hours
- Attendance percentage

## Leave Management

### Apply for Leave

1. Go to Leave Management
2. Click "Apply Leave"
3. Fill in the form:
   - **Leave Type**: Casual, Sick, Annual
   - **Start Date**: First day of leave
   - **End Date**: Last day of leave
   - **Reason**: Explain why you need leave
   - **Attachments**: Upload supporting documents (optional)
4. Click "Submit"

### Leave Types

- **Casual Leave**: 12 days per year
- **Sick Leave**: 10 days per year
- **Annual Leave**: 20 days per year
- **Maternity Leave**: As per policy
- **Paternity Leave**: As per policy

### Check Leave Balance

1. Go to Leave Management
2. View "Leave Balance" card
3. See available days for each leave type

### View Leave Applications

1. Go to Leave Management
2. See all your applications with status:
   - **Pending**: Awaiting approval
   - **Approved**: Leave granted
   - **Rejected**: Leave denied
   - **Cancelled**: You cancelled the application

### Cancel Leave Application

1. Go to Leave Management
2. Find the pending leave
3. Click "Cancel"
4. Confirm cancellation

**Note:** Only pending leaves can be cancelled.

## Troubleshooting

### Cannot Check In

**Issue:** Check-in button is disabled

**Solutions:**
- You may have already checked in
- Check if you're on leave
- Contact HR if issue persists

### Fingerprint Not Working

**Issue:** Fingerprint verification fails

**Solutions:**
- Ensure finger is clean and dry
- Try another enrolled finger
- Re-enroll fingerprint if needed
- Use manual check-in as backup

### Leave Application Error

**Issue:** Cannot submit leave application

**Solutions:**
- Check if you have sufficient leave balance
- Ensure dates are valid
- Verify reason is provided
- Check for overlapping leave applications

### AI Assistant Not Responding

**Issue:** Chatbot doesn't answer

**Solutions:**
- Check internet connection
- Refresh the page
- Try rephrasing your question
- Contact IT support

## Frequently Asked Questions

**Q: Can I check in from home?**
A: Yes, but it will be marked as Work From Home if enabled.

**Q: What happens if I forget to check out?**
A: Auto check-out will happen at 6:00 PM if enabled.

**Q: Can I apply leave for past dates?**
A: No, leave must be applied in advance.

**Q: How do I know if my leave is approved?**
A: You'll receive an email notification and can check status in Leave Management.

**Q: Can I cancel approved leave?**
A: Contact your manager or HR for approved leave cancellation.
